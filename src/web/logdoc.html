Log message documentation

XINS provides extensive log documentation. Each message logged by XINS has a unique identifier and is documented using a system called Logdoc.

Log documentation is available for all 3 major components of the XINS/Java framework:

  • XINS/Java Common Library
  • XINS/Java Client Framework
  • XINS/Java Server Framework

Note that each of these components maps to one JAR file.
